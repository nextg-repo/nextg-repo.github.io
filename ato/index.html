AI아토뷰어 소개

AI아토뷰어는 최신 생성형 AI 기능과 OCR기능을 활용하여 화면에 표시되는 사진, 그림, 그래프 등의 이미지를 시각장애인 사용자가 이해할 수 있도록 해설하며, 생성형 AI 기능을 활용하여, 사용자의 요구에 따라 각종 전문 분야에 관한 답변을 문서로 생성할 수 있는 디바이스 입니다.

아토 이미지 해설

  • AI아토뷰어는 웹 서핑중 만나게 되는 화면 상의 이미지를 해설해 줍니다.
  • AI아토뷰어는 보관 중인 이미지 파일을 화면에 실행하여 해당 이미지를 해설할 수 있습니다.
  • AI아토뷰어는 화면 상에 나타나는 각종 그래프에 대한 해설을 들을 수 있습니다.
  • AI아토뷰어는 이미지에 대한 전반적인 설명을 확인한 후 추가적인 질문을 통해 좀 더 세밀한 정보를 얻을 수 있습니다.

아토 어시스턴트

  • 아토 어시스턴트는 작가, 컴퓨터 튜터, 의료 상담, 심리 상담, 법률 상담, 세무 상담, 역사 해설가 등의 어시스턴트 카테고리를 통해 사용자가 언제든지 질문하고 답변을 받을 수 있습니다.
  • 아토 어시스턴트는 추가적인 질문을 통해 전문 분야에 대한 심도 있는 답변을 문서로 작성할 수 있습니다.
  • 아토 어시스턴트는 전문적인 답변 내용을 텍스트 파일로 저장하여 다른 업무와 학습에 활용할 수 있습니다.

아토 이미지 해설 활용 방법

아토 이미지 해설 기능은 현재 화면에 표시되는 이미지를 해설하는 기능 입니다.

아토 이미지 해설 프로그램 화면 구조

  • 아토 이미지 해설 프로그램은 최상단에 제목표시줄이 위치하며, 제목 표시줄에 'AI아토뷰어'라고 타이틀이 적혀 있습니다.
  • 제목표시줄 아래에는 주메뉴 표시줄이 위치하며, 파일, 이미지, 분석, 도움말 등의 4가지 주메뉴가 위치 합니다.
  • 주메뉴표시줄 아래에는 질문을 입력할 수 있는 편집창과 질문에 대한 분석을 요청하는 요청 버튼이 위치 합니다.
  • 질문 입력 편집창 아래에는 분석 결과를 보여주는 멀티라인 편집창이 위치 합니다.
  • 질문 입력 편집창과 분석 결과 멀티라인 편집창은 Ctrl+Tab 키를 눌러 이동할 수 있습니다.
  • 분석 결과 멀티라인 편집창에서 질문 입력 편집창으로 이동할 때에는 Alt+D 키를 눌러 이동할 수 있습니다.
  • 질문 입력 편집창에서 질문 입력 후 Enter 키를 누르면 답변 요청이 실행 됩니다.
  • 질문 입력창에서 여러 줄로 질문을 입력할 때 줄 바꿈 단축키는 Ctrl+Enter 키 입니다.

이미지 파일 해설 방법

  1. 이미지 해설이 필요한 이미지 파일을 찾아 실행 합니다.
  2. Ctrl+Alt+I 키를 눌러 아토 이미지 해설 프로그램을 실행 합니다.
  3. 아토 이미지 해설 프로그램이 실행되면, 별도의 지시어 없이 엔터키를 누르면 현재 이미지에 대한 전반적인 해설을 위해 분석을 시작 합니다.
  4. 잠시 후 '사용자'라는 멘트가 들리면 이미지 분석이 완료 된 상태이며 스크린리더가 동작 중인 경우 아래 방향키나 연속읽기 기능을 통해 분석 결과를 확인할 수 있습니다.

추가 질문 하기

  • 현재 이미지에 대한 해설을 확인한 후 추가적인 질문을 통해 좀 더 세부적인 답변을 원하시면 Alt+D 키를 눌러 질문 입력 편집창으로 이동 합니다.
  • 질문 입력 편집창에서 추가 질문 내용을 입력한 후 엔터키를 누르면 추가 분석이 시작 됩니다.
  • 잠시 후 사용자라는 멘트가 들리면 스크린리더가 실행된 상태에서는 아래 방향키나 연속읽기 기능을 통해 추가 분석 내용을 확인할 수 있습니다.
  • 이후 더 추가 질문이 있으면 위 과정을 반복해서 실행하면 됩니다.

전체 화면 분석

  • 아토 이미지 해설 기능을 실행할 때 현재 화면에 보이는 모든 내용을 한꺼번에 분석하는 방법 입니다.
  • 아토 이미지 해설 프로그램의 주메뉴에서 이미지 메뉴의 하위 메뉴 중 전체 화면 캡쳐 메뉴항목을 선택한 후 이미지 해설을 진행하면 됩니다.
  • 전체 화면 해설을 선택하기 위한 기능키는 Alt+F 입니다.

활성 윈도우 이미지 해설

  • 현재 화면에 여러 개의 프로그램 창이 겹쳐서 실행되어 있는 경우 이 중 활성화 되어 있는 윈도우만 이미지 분석을 실행할 때 활용 합니다.
  • 아토 이미지 해설 프로그램의 주메뉴에서 이미지 메뉴의 하위 메뉴 중 윈도우 화면 캡쳐 메뉴항목을 선택한 후 이미지 해설을 진행하면 됩니다.
  • 활성 윈도우 해설을 선택하기 위한 기능키는 Alt+W 입니다.

도표 해설

  • 아토 이미지 해설을 통해 그래프와 같은 이미지를 분석할 때 도표 해설 메뉴를 활용하면 특별한 질문 입력 없이 도표에 대한 좀 더 구체적인 해설을 확인할 수 있습니다.
  • 첫째 분석하고 싶은 그래프를 화면에 나타나게 합니다.

  • 도표 해설 메뉴항목을 실행하는 기능키는 Alt+C 입니다.

인물 해설

  • 첫째 분석하고 싶은 인물 이미지를 화면에 나타나게 합니다.
  • 둘째 Ctrl+Alt+I 키를 눌러 아토 이미지 해설 프로그램을 실행 합니다.
  • 셎째 아토 이미지 해설 프로그램의 주메뉴 중 분석 메뉴의 하위 메뉴인 인물 해설 메뉴를 찾아 실행하면 됩니다.

    4. 저장 메뉴가 실행되면 저장 대화상자가 나타나고 이 대화상자에서 저장 위치와 저장할 파일 이름을 지정하고 저장 합니다.

    <참고> 저장되는 질문과 답변의 내용은 카테고리에 상관 없이 아토 어시스턴트가 실행된 상태에서 작성된 질문과 답변 내용 전체이며, 만약 아토 어시스턴트 프로그램 창을 종료한 경우 과거에 작성했던 질문과 답변은 저장할 수 없습니다.
