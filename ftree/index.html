숲나무 메뉴얼

소개

용도

  • '숲나무'는 저시력인이 Windows PC를 활용할 때, PC 화면에서 주시하는 지점을 별도의 디스플레이 장치에 확대하여 표시해 주는 솔루션입니다.
  • PC 화면의 전체 이미지와 필요한 지점을 별도의 장치에서 실시간으로 확대 출력하므로 화면의 전체와 부분을 동시에 볼 수 있는 장점이 있습니다.

주요 기능

  • 사용자가 PC화면에서 주시하고자 하는 지점으로 마우스포인터나 키보드 초점을 이동할 때마다 해당 영역이 별도의 디바이스 장치에서 확대되어 출력됩니다.
  • 숲나무 디바이스에서 출력되는 확대 비율은 1배에서 50배 이상입니다.
  • 숲나무 디바이스에서 출력되는 화면 효과는 '원본 화질', '화질 보정', '색상 반전', '그레이 색상'이 있으며 수시로 전환할 수 있습니다.
  • 숲나무 디바이스의 음성 인식 기능을 통해 PC의 입력 UI에서 육성으로 글자를 입력할 수 있습니다.
  • 숲나무 디바이스의 음성 합성 기능을 통해 PC 화면에 대량으로 출력된 텍스트를 TTS 음성으로 읽을 수 있습니다.
  • 문서의 문장을 읽을 때, '숲나무 디바이스'에서 읽는 방향으로 자동 스크롤합니다.

특징

  • '숲나무'는 전체와 부분을 동시에 주시할 수 있어, 불필요한 동작(빈번한 확대 온/오프, 초점 이동 등)과 화면 왜곡(영역 누락, 화면 비율 등)으로 인한 오인이 없습니다.
  • 이와 같이 노이즈 현상과 동작이 없기 때문에 눈의 피로도가 현저히 적습니다.
  • 또한 음성 인식과 음성 합성 기능이 결합되어 효율적인 활용이 가능합니다.

초기 설치 및 설정

Windows용 '숲나무 애플리케이션' 설치

  • 숲나무 패키지에 제공된 USB를 대상 Windows PC에 연결합니다.
  • USB 드라이브에서 install.exe 파일을 실행하여 '숲나무 애플리케이션'을 설치합니다.
  • Windows 바탕화면에 '숲나무' 아이콘이 생성되면 설치가 정상적으로 완료된 것입니다.

'숲나무 디바이스' 초기 설정

  • '숲나무 디바이스'에서 '설정' -> '연결' -> 'WI-FI' 메뉴를 이용하여 자신의 환경에 맞는 와이파이로 연결합니다.
  • '숲나무 디바이스'에서 '설정' -> '디스플레이' -> '화면 자동 꺼짐 시간' 메뉴를 이용하여 '없음'으로 적용합니다. (기본 적용되어 있음)
  • '숲나무 디바이스'에서 '설정' -> '잠금 화면' -> '화면 잠금 방식' 메뉴를 이용하여 '설정 안 함'으로 적용합니다. (기본 적용되어 있음)

초기 권한 설정

  • 숲나무는 PC에서 Windows용 숲나무 애플리케이션과 숲나무 디바이스 간 초기 연결시 '권한/연결 허용'에 관한 승인 여부를 받을 수 있으며, 이때 반드시 '허용' 버튼을 눌러야 합니다.

기본 사용법

유선 연결

대기

  • Windows PC와 '숲나무 디바이스'를 USB 케이블로 서로 연결합니다.
  • 1분 내외로 '숲나무 디바이스'에서 '디디어 또는 USB 디버깅 연결' 대화상자가 뜨는데, 이때 '허용' 버튼을 누릅니다.
  • Windows에서 '숲나무 아이콘'을 실행합니다.
  • 숲나무 메인화면에서 '유선 연결' 버튼을 누릅니다.

연결

  • '숲나무 디바이스'에서 '숲나무 아이콘'을 실행합니다.
  • 숲나무 매인화면에서 '돋보기 이미지'를 누릅니다.
  • Windows 화면이 '숲나무 디바이스'에서 출력되면 정상적으로 연결된 것입니다.

연결 해제

  • 숲나무 해제/종료 방법은 두가지가 있습니다.
  • 첫번째 방법은 '숲나무 디바이스'에서 하단 우측에 있는 '취소/뒤로가기' 버튼을 눌러 종료/해제할 수 있습니다.
  • 두번째 방법은 Windows에서 '숲나무 아이콘'을 실행 한 후, 매인화면에서 '유선 연결 종료' 버튼을 눌러 종료/해제할 수 있습니다.

무선 연결

대기

  • Windows에서 '숲나무 아이콘'을 실행합니다.
  • 숲나무 매인화면에서 '무선 연결' 버튼을 누릅니다.
  • 내부 IP 주소가 표시되는데, 기억해 둡니다.

연결

  • '숲나무 디바이스'에서 '숲나무 아이콘'을 실행합니다.
  • 돋보기 이미지를 3초간 길게 누릅니다.
  • IP 주소 입력창이 뜨는데, 이때 기억해 둔 IP 주소를 입력하고 '연결'버튼을 누릅니다.
  • 상기의 작업은 한번만 수행하면 됩니다.
  • 이후부터는 돋보기 이미지를 길게 누르지 않고 바로 터치만으로 연결됩니다.
  • Windows 화면이 '숲나무 디바이스'에서 출력되면 정상적으로 연결된 것입니다.

해제

  • 숲나무 해제/종료 방법은 두가지가 있습니다.
  • 첫번째 방법은 '숲나무 디바이스'에서 하단 우측에 있는 '취소/뒤로가기' 버튼을 눌러 종료/해제할 수 있습니다.
  • 두번째 방법은 Windows에서 '숲나무 아이콘'을 실행 한 후, 매인화면에서 '무선 연결 종료' 버튼을 눌러 종료/해제할 수 있습니다.

줌 모드

기능 호출

  • PC 키보드에서 왼쪽 CTRL + WIN + F9 키를 동시에 누르면 '숲나무 디바이스'가 '줌 모드'로 실행 또는 전환됩니다.
  • '줌 모드'는 '숲나무'의 초기 연길 시 항상 기본 상태로 동작합니다.

배율

  • PC 키보드에서 왼쪽 CTRL + WIN + KEYPAD Plus 키를 동시에 누르면 '숲나무 디바이스'의 화면 이미지가 확대됩니다.
  • PC 키보드에서 왼쪽 CTRL + WIN + KEYPAD Minus 키를 동시에 누르면 '숲나무 디바이스'의 화면 이미지가 축소됩니다.
  • '숲나무 디바이스'에 터치스크린을 두 손가락을 대고 벌리거나 오므릴 때 마다 확대/축소됩니다.

화질

  • PC 키보드에서 왼쪽 CTRL + WIN + F5 키를 동시에 누르면 '숲나무 디바이스'의 화면 이미지가 일반 화질로 표시됩니다.
  • PC 키보드에서 왼쪽 CTRL + WIN + F6 키를 동시에 누르면 '숲나무 디바이스'의 화면 이미지가 고화질로 표시됩니다.
  • PC 키보드에서 왼쪽 CTRL + WIN + F7 키를 동시에 누르면 '숲나무 디바이스'의 화면 이미지가 반전 색상으로 표시됩니다.
  • PC 키보드에서 왼쪽 CTRL + WIN + F8 키를 동시에 누르면 '숲나무 디바이스'의 화면 이미지가 그레이 색상으로 표시됩니다.
  • '숲나무 디바이스'의 터치스크린을 세 손가락으로 길게 누를 때마다 '일반 화질', '화질 보정', '색상 반전', '그레이 색상'으로 전환됩니다.

탐색 및 화면 재생

  • Windows에서 마우스를 이동하면 포인터가 가리키는 해당 지점이 '숲나무 디바이스'에 확대되어 실시간으로 표시됩니다.
  • Windows의 모든 화면 변화에 대하여 사용자 조작이 없어도 '숲나무 디바이스'에 확대되어 실시간으로 표시됩니다.
  • '숲나무 디바이스'의 터치스크린를 한 손가락으로 접촉하여 탐색하거나 객체를 두번 탭하여 실행할 수 있습니다.

키보드 초점 추적

  • Windows에서 Tab키 또는 방향키로 초점을 이동시킬 때마다 변화 지점을 자동 추적하여 '숲나무 디바이스'에 실시간으로 표시됩니다.
  • Windows에서 글자를 타이핑할 때마다 입력 중인 내용을 자동 추적하여 '숲나무 디바이스'에 실시간으로 표시됩니다.
  • 키보드 초점 및 커서 추적 기능을 켜거나 끄려면 PC 키보드의 CTRL + WIN + F11 키를 동시에 눌러 전환할 수 있습니다.

마우스패드 모드

기능 호출

  • PC 키보드에서 왼쪽 CTRL + WIN + F10 키를 동시에 누르면 '숲나무 디바이스'가 '마우스패드 모드'로 실행 또는 전환됩니다.

탐색

  • '숲나무 디바이스'의 터치스크린에서 한 손가락을 댄 다음 움직이면 화면상의 절대 좌표에 맞추어 Windows 마우스 포인터의 위치가 매칭되어 이동합니다.

객체 실행 및 팝업 메뉴 호출

  • '숲나무 디바이스'의 터치스크린을 마우스패드로 사용하는 중에 한 손가락으로 두번 탭하면 Windows의 마우스포인터가 가리키는 객체를 실행합니다.
  • '숲나무 디바이스'의 터치스크린을 마우스패드로 사용하는 중에 한 손가락으로 길게 터치하면 Windows의 마우스포인터가 가리키는 객체에 대한 팝업 메뉴가 호출됩니다.

음성 입력

사전 요건

  • 이 기능은 '숲나무 디바이스'가 WI-FI에 연결되어 있어야 합니다.
  • 이 기능을 활용하기 위해 Windows의 메모장, 입력창, 검색라인 등 입력 UI에 먼저 키보드 초점을 보냅니다.

기능 호출

  • PC에서 메모장 등의 입력 UI에 포커스된 상태에서, PC 키보드 CTRL + WIN + KEYPAD Slash 키를 동시에 누르면 '숲나무 디바이스'에 신호음이 출력되는 데 이 때 육성으로 발화하면 해당 내용이 PC 입력창에 표시됩니다.
  • 또는 키보드 단축키 대신 '숲나무 디바이스'에서 두 손가락 두번 탭으로도 동일하게 작동합니다.

음성 출력

사전 요건

  • 이 기능을 활용하기 위해 Windows의 워드 문서, 웹 문서 등에 포함된 문장을 먼저 선택(블럭)합니다.

기능 호출

  • PC에 문장이 선택된 상태에서 키보드 CTRL + WIN + KEYPAD Star(별표) 키를 동시에 누르면 해당 문장이 '숲나무 디바이스'에서 음성 출력됩니다.
  • 또는 키보드 단축키 대신 '숲나무 디바이스'에서 두 손가락을 길게 눌러도 동일하게 작동합니다.

포인터 오토 스크롤

기능

  • 포인터 오토 스크롤은 PC 화면에서 가리키고 있는 지점에서부터 가로측의 오른쪽 방향으로 시점을 자동 이동시키면서 '숲나무 디바이스'에 반영되어 표시되는 기능입니다.
  • 이 기능을 통해 사용자는 시선을 고정해도 되기 때문에 문장을 편하게 읽을 수 있습니다.

기능 호출

  • PC 키보드 SHIFT + CAPSLOCK 키를 동시에 누르면 기능이 게시됩니다.
  • 동작 중인 기능을 해제할 때도 같은 키를 누릅니다.

조작

  • 이 기능은 화면 오른쪽 끝 지점에 도달하면 자동으로 화면 왼쪽 처음으로 자동 이동하며, 0.5초간 머물다가 다시 오른쪽으로 이동합니다.
  • 사용자는 마우스포인터로 상하좌우 이동에 개입할 수 있습니다.
  • 문장이 화면 끝 지점보다 짧아 더 이상의 이동이 무의미할 때는 마우스 좌측버튼을 클릭하여 화면 왼쪽 처음 지점으로 곧장 이동할 수 있습니다.
  • 마우스를 우측으로 강제하면 이동 속도가 점차 빨라집니다.
  • 마우스를 좌측으로 강제하면 이동 속도가 점차 느려집니다.

라인 포인터

기능

  • 라인 포인터 기능은 PC 화면에 빨간색의 가로선이 표시되는 기능입니다.
  • 이 기능을 통해 사용자는 주시하는 문장의 위치를 직관적으로 파악할 수 있어, 시선의 안정감을 줍니다.

기능 호출

  • PC 키보드 CTRL + WIN + F12 키를 동시에 누르면 기능이 게시됩니다.
  • 동작 중인 기능을 해제할 때도 같은 키를 누릅니다.

조작

  • 마우스를 상하로 움직이면 빨간색의 가로선이 위아래로 따라 이동합니다.
  • 마우스를 좌우로 움직이면 빨간색 가로선에 포함된 노란색 작은 막대가 방향을 따라 이동합니다.

자주 묻는 질문

  • '숲나무'는 Windows 버전에 제약이 있습니까?

    '숲나무'는 Windows 10, 11 버전에서 정상 작동합니다.

  • 공인(외부) IP 주소로 무선 연결이 가능합니까?

    '숲나무'는 공인 IP, 내부 IP 모두에서 접속이 가능합니다.

  • IP 주소를 매번 기억하여 접속해야 합니까?

    '숲나무'는 한번 접속이 성공한 IP 주소는 기록되어 있기 때문에 이후 부터는 자동 접속됩니다. 다만, 새로운 PC와 연결할 때는 다시 IP 주소를 입력해야 합니다.

  • 얼마나 확대됩니까?

    '숲나무'는 이론상 무제한 확대됩니다. 다만 통상 20배 내외에서 사용하기를 권해 드립니다.

  • 음성 인식이 잘 안될 때는 어떻게 조치합니까?

    '숲나무'의 음성 인식은 '숲나무 디바이스'가 WI-Fi에 연결되어 있어야만 정상 작동합니다.

  • '숲나무 디바이스'를 스마트폰으로 대체하여 사용할 수 있습니까?

    '숲나무 디바이스'에 포함된 '숲나무 애플리케이션'은 '숲나무 디바이스'이 외에서는 작동하지 않기 때문에 스마트폰으로 대체할 수 없습니다.

  • '숲나무 디바이스'의 화면 이동이 느립니다. 어떻게 조치할 수 있습니까?

    '숲나무 디바이스'는 PC와 상호 데이터를 실시간으로 교환하며 작동하기 때문에 연결 환경이 성능에 영향을 미칩니다. '무선 연결'에서 이런 증상이 지속되면, '유선 연결'을 사용하거나, 공유기 상태를 점검해 보시기 바랍니다.
