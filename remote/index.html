마이원리모트 소개

  • 마이원리모트는 원격지의 PC를 모바일 마이원리모트 기기에서 제어할 수 있습니다.
  • 마이원리모트는 모바일 기기에 연결된 물리적 키보드를 활용하여 시각장애인 사용자가 원격지의 PC를 제어할 수 있습니다.
  • 마이원리모트는 가상키보드를 활용하여 원격지 PC를 제어할 수 있습니다.
  • 마이원리모트는 윈도우용 마이원리모트 프로그램에 탑재된 NVDA를 활용하여 시각장애인 사용자가 원격지에서 PC를 스크린리더를 사용하여 제어할 수 있습니다.
  • 마이원리모트는 필요시 원격지 PC의 화면을 캡처하여 모바일 디바이스에서 확인할 수 있습니다.
  • 마이원리모트는 원격지 PC에 저장된 음원파일이나 동영상파일들을 스트리밍 방식으로 모바일 디바이스에서 재생할 수 있습니다.

마이원리모트 계정 등록

  • 마이원리모트를 사용하기 위해서는 반드시 계정이 필요 합니다.
  • 마이원리모트 계정을 생성하기 위해서는 반드시 고객센터로 전화하여 계정 등록을 유선으로 진행 하셔야 합니다.

  • 처음 계정을 생성하면 임시 비밀번호가 부여되며, 반드시 사용자가 비밀번호를 변경해야만 마이원리모트 사용이 가능 합니다.

공유기 포트포워딩 설정

  • 공유기 포트 포워딩이란 공유기를 사용하는 경우 외부에서 공유기를 통해 PC에 접속할 때 활용할 포트를 지정하여 내부 IP를 통해 외부와 PC를 연결하는 설정 입니다.
  • 마이원리모트를 사용하기 위해서는 원격제어용 포트와 FTP용 포트 등 총 2개의 포트가 필요 합니다.
  • 일반적인 공유기 포트 포워딩 방법은 아래와 같습니다.
    1. 공유기 포트 포워딩 설정을 위해 먼저 PC의 cmd창에 ipconfig를 입력하여 나타난 결과 중 IPv4 주소와 기본 게이트웨이 주소를 확인해 둡니다.
    2. 공유기 설정 메뉴로 진입하여 포트포워딩 메뉴를 찾아 실행 합니다.
    3. 포트포워딩 등록화면에서 포트포워딩 프로필 정보를 입력해야 하는데, 프로필 이름 입력, 프로토콜 선택, 내부 IP 주소 입력, 외부 포트 입력, 내부 포트 입력이 있습니다.
    4. 프로필 이름은 특별한 제약이 없으니, 간편한 이름으로 기재합니다.
    5. 프로토콜 선택은 반드시 'TCP' 로 선택합니다.
    6. 내부 IP 주소는 사전에 확인해 둔 IPv4 주소를 기재합니다.
    7. 외부 포트는 범위를 지정하여 기재하는데, 특별한 경우가 아니라면 9998에서 9999로 지정 합니다.
    8. 내부 포트도 범위를 지정하여 기재하는데, 반드시 상기 외부 포트 번호와 동일해야 합니다.
    9. '프로필 정보가 모두 기재되었다면 '추가' 또는 '등록' 버튼을 누릅니다.
    10. 포트 등록 화면 하단에 임시(?) 등록된 프로필 항목이 표기되는데, 이 항목을 선택 합니다.
    11. 최종 '저장' 또는 '적용' 버튼을 찾아 누릅니다.

<참고> 공유기 제품의 종류에 따라 포트포워딩 설정 방법은 위에서 제시한 과정과 다를 수 있습니다.

윈도우용 마이원리모트 프로그램 설치

  • 윈도우용 마이원리모트를 설치 하려면 다음과 같은 단계로 진행 합니다.
    1. 제공된 USB메모리를 PC에 연결합니다.
    2. 내PC 아이콘을 실행하여 드라이브 목록에서 마이원리모트 항목을 찾아 엔터키를 눌러 실행 합니다.
    3. 마이원리모트 설치 파일 목록에서 install.exe 파일을 찾아 엔터키를 눌러 실행 합니다.
    4. 설치 대화상자가 나타나면 엔터키를 눌러 설치를 진행 합니다.
    5. 설치가 완료되면 엔터키를 누르면 설치 대화상자가 사라지고 완료 됩니다.
  • 마이원리모트가 설치되면 바탕화면에 MyoneRemote 아이콘이 생성 됩니다.

모바일 마이원리모트 앱 설치

  • 모바일 마이원리모트를 설치하기 위해서는 반드시 애플 계정이 필요합니다.
  • 제공된 모바일 마이원리모트 디바이스에 Apple계정을 등록한 후 앱스토어에 방문하여 마이원리모트 앱을 검색하여 설치 합니다.

유의사항.

  • 마이원리모트는 원격제어할 PC가 위치한 네트워크 환경이 방화벽이나 포트 포워딩 설정이 어려울 경우 외부 연결이 어려울 수 있습니다.
  • 마이원리모트는 한 번에 한 대의 PC와만 연결이 가능 합니다.
  • 마이원리모트는 모바일 마이원리모트 디바이스에서 원격지에 접속한 PC를 제어할 때 NVDA스크린리더를 조작하는 방법으로 제어 합니다. 따라서 원격지 PC에 설치된 스크린리더와는 별도의 방식으로 동작 합니다.
  • 마이원리모트의 계정 생성은 반드시 고객센터로 문의 하시기 바랍니다.
  • 원격제어를 위한 PC가 절전모드나 로그온 상황에서는 원격제어가 불가능 합니다.
  • 모바일 마이원리모트의 텍스트뷰, 이미지뷰, 가상키보드뷰 상황에서는 보이스오버를 종료한 후 사용해야 하며, FTP뷰 상황에서는 보이스오버를 실행한 상태에서 사용해야 합니다.

윈도우용 마이원리모트 프로그램

  • 윈도우용 마이원리모트 프로그램은 원격제어를 실행할 PC에 설치 합니다.
  • 원격제어가 원할하게 수행되려면 PC가 네트워크에 연결된 상태에서 원격제어가 활성화 되어 있어야 합니다.

원격제어 활성화

  • 윈도우용 마이원리모트 프로그램 창에서 원격제어 활성화 버튼을 찾아 스페이스키를 누르면 PC는 원격제어를 수행할 준비를 시작 합니다.
  • 원격제어가 활성화 되면 '원격제어는 "9999" 포트로, FTP는 "9998" 포트로 활성화되었습니다.'와 같은 대화상자가 나타나며, 엔터키를 누르면 원격제어 활성화가 완료 됩니다.
  • 원격제어 활성화가 완료되면 원격제어 활성화 버튼은 원격제어 종료 버튼으로 변경 됩니다.
  • 원격제어 종료 버튼에서 스페이스키를 누르면 'MyoneRemote가 원격 대기 모드에서 완전히 종료되었습니다.'라는 대화상자가 나타나고, 엔터키를 누르면 원격제어는 종료 됩니다.

비밀번호 변경

  • 비밀번호 변경 버튼을 누르면 비밀번호 설정을 위한 대화상자가 나타납니다.
  • 첫번째 편집창에는 변경할 계정의 아이디를 입력 한 후 탭 키를 눌러 다음 편집창으로 이동 합니다.
  • 두번째 편집창은 기존 비밀번호를 입력하는 편집창이며, 이 곳에 기존에 사용하던 비밀번호를 입력한 후 탭 키를 눌러 다음 편집창으로 이동 합니다.
  • 세번째 편집창은 변경할 비밀번호 입력을 위한 편집창으로, 이 곳에 새로운 비밀번호를 입력 한 후 탭 키를 눌러 확인 버튼으로 이동 합니다.
  • 확인 버튼을 누르면 비밀번호 변경이 완료 됩니다.

<참고> 마이원리모트를 처음 설치하여 임시로 부여 받은 비밀번호는 반드시 변경하여 사용해야 하며, 비밀번호 변경 작업을 수행하면 해당 PC에 마이원리모트 서비스 계정은 자동 등록 됩니다.

포트번호 변경

  • 마이원리모트는 원격제어와 FTP 활용을 위해 2개의 포트를 사용 합니다.
  • 기본적으로 마이원리모트는 원격제어의 경우 9999포트를 사용하며, FTP의 경우 9998포트를 사용 합니다.
  • 특별한 경우가 아니라면 굳이 포트 번호를 변경할 필요는 없으며, 아주 예외적인 경우에만 포트 번호를 변경하여 사용 합니다.
  • 이 곳에서 포트 번호를 변경할 경우 반드시 공유기의 포트 포워딩과 모바일 마이원리모트의 포트 번호와 일치 되도록 설정 하여야 합니다.
  • 포트 변경 버튼을 누르면 '각각의 포트 변호를 9000에서 9999 범위에서 입력해 주세요.'라는 대화상자가 나타납니다.
  • 이 대화상자의 첫 번째 편집창은 원격제어 포트를 입력하는 편집창이며, 이 곳에 9000번에서 9999까지의 번호 중 하나의 번호를 입력 합니다.
  • 이 대화상자의 두 번째 편집창은 FTP포트를 입력하는 편집창이며, 이 곳에 9000번에서 9999까지의 번호 중 하나의 번호를 입력 합니다.
  • 원격제어 포트와 FTP 포트 번호가 동일하게 설정되지 않도록 주의 합니다.

<참고> 윈도우 마이원리모트 프로그램에서 포트 번호를 변경하면 공유기 포트포워딩과 모바일 마이원리모트의 포트 번호도 함께 변경해야만 합니다.

시작프로그램 등록

  • 윈도우 부팅시에 자동으로 마이원리모트가 활성화 되도록 시작 프로그램에 등록하는 설정 입니다.
  • 시작프로그램 등록 버튼을 찾아 스페이스 키를 눌러 실행 합니다.
  • 'MyoneRemote가 시작프로그램에 등록되었습니다. '라는 대화상자가 나타나면 엔터키를 눌러 완료 합니다.
  • 시작프로그램 등록이 끝나면 해당 버튼은 시작프로그램 해제 버튼으로 변경 됩니다.
  • 시작프로그램 해제 버튼을 누르면 'MyoneRemote가 시작프로그램에서 해제되었습니다.'라는 대화상자가 나타나고 엔터키를 누르면 완료 됩니다.
  • 마이원리모트가 시작프로그램에 등록되면 PC 부팅시 자동으로 마이원리모트가 실행되고 원격제어가 활성화 됩니다.

모바일 마이원리모트 앱

모바일 마이원리모트 앱 구조 및 주요 기능

  • 모바일 마이원리모트 앱을 설치한 후 처음 실행하면 로그인 화면이 나타납니다.
  • 승인 받은 계정 정보를 입력한 후 로그인을 실행하면 마이원리모트 앱의 홈 화면이 나타납니다.
  • 모바일 마이원리모트 앱은 화면 최상단에 마이원리모트라는 머리말이 표시되며, 그 아래에 로그인된 계정의 아이디가 표시되고, 그 아래에 연결 버튼, 가상키보드로연결 버튼, 설정 버튼, 로그아웃 버튼 등의 순서로 항목들이 표시 됩니다.

원격 연결

  • 모바일 마이원리모트와 원격지 PC를 연결 하기 위해서는 먼저 원격지 PC의 마이원리모트 프로그램이 원격제어 활성화 상태가 되어 있어야 합니다.
  • 모바일 마이원리모트 앱의 연결 버튼을 누르면 원격지 PC와 연결이 시도되고 연결이 성공되면 텍스트뷰 상황으로 연결 됩니다.

가상키보드로 원격 연결

  • 가상키보드로 연결되면 원격지 PC를 제어할 때 모바일 마이원리모트에 물리적 키보드가 연결되어 있지 않아도 화면에 표시되는 가상키보드를 통해 제어가 가능 합니다.
  • 가상키보드로 연결되면 모바일 마이원리모트 화면 하단에 가상의 키보드가 표시되며, 보이스오버를 종료한 상태에서 손가락을 키보드에 위치하면 해당 키보드의 명칭을 음성으로 안내하고 손가락을 떼면 해당 키보드가 동작 합니다.
  • 만약 여러 개의 키보드를 조합하여 눌러야 할 경우 고정키에 해당하는 키는 3초간 누르고 있으면 고정 됩니다.
    • 예를 들어 Win+M 이라는 조합키를 누르려면 먼저 Win키를 찾아 3초간 누르면 고정 이라는 음성이 들리면 손을 떼고 M키를 차자 눌렀다 떼면 Win+M의 조합키로 동작 합니다.

설정

  • TTS설정은 모바일 마이원리모트의 음성 설정을 의미하며, 아이패드의 보이스오버 설정과는 무관 합니다.
  • TTS설정으로 진입하면, TTS속도와 TTS종류를 선택하는 메뉴가 표시 됩니다.
  • TTS종류는 보이스오버 설정에서 설치한 TTS엔진들만 목록에 표시 됩니다.
  • TTs속도나 TTS종류를 변경한 후 반드시 화면 하단에 저장 버튼을 눌러야 변경된 설정이 적용 됩니다.
  • 텍스트설정은 텍스트뷰 상태로 원격 PC와 연결 되었을 때 모바일 마이원리모트 화면에 표시되는 텍스트의 크기를 지정하는 설정 입니다.
  • 텍스트크기 설정을 변경한 후 화면 하단의 저장 버튼을 눌러야 설정이 완료 됩니다.
  • 포트설정은 원격지 PC와 연결할 때 사용하는 포트 번호를 설정하며, 원격제어 포트와 FTP 포트로 나누어서 설정할 수 있습니다.
  • 포트 변경시에는 반드시 9000부터 9999까지의 숫자만을 사용하며, 원격제어 포트 번호와 FTP 포트번호가 동일하지 않도록 주의 합니다.
  • 만약 모바일 마이원리모트에서 포트 번호를 변경하면 반드시 공유기 포트포워딩 설정과 윈도우용 포트 설정도 동일하게 변경해야 합니다.

마이원리모트 활용하기

원격제어 실행 및 종료

원격제어를 활용하기 위해서는 반드시 사전에 공유기 포트포워딩, 계정 승인 등의 사전 준비가 완료 되어야 합니다.

원격제어를 실행하는 과정은 다음과 같습니다.

  1. 원격지 PC에서 마이원리모트 프로그램을 실행한 후 원격제어 활성화 버튼을 눌러 원격제어 연결 대기 상태를 만들어 둡니다.
  2. 모바일 마이원리모트 앱을 실행하여 로그인 합니다.
  3. 모바일 마이원리모트 앱에서 연결 버튼을 누르면 원격제어가 실행 됩니다.
  4. 모바일 마이원리모트의 홈 화면으로 이동하면 원격제어는 대기 상태가 됩니다.
  5. 모바일 마이원리모트를 앱 전환기에서 종료하면 원격지PC와 연결이 완전 종료 됩니다.

<참고> 모바일 마이원리모트가 활성화 되어 있어도 원격지 PC의 마이원리모트 프로그램의 원격제어가 종료 되면 원격제어는 종료 됩니다.

각종 뷰모드 활용 방법

  • 모바일 마이원리모트에서는 텍스트뷰, 이미지뷰, FTP뷰, 가상키보드뷰와 같이 다양한 뷰 모드를 제공 합니다.
  • 가상키보드 뷰모드를 제외 하고 나머지 텍스트 뷰모드, 이미지 뷰모드, FTP 뷰모드 간에는 언제든 단축키와 화면 하단의 버튼을 통해 상호 전환이 가능 합니다.
  • 모바일 마이원리모트 앱에서 연결을 실행 하면 기본적으로 텍스트 뷰 모드로 연결되며, 텍스트 뷰 모드에서는 원격지 PC를 조작하는 모든 과정이 화면에 텍스트로 표시되며 TTS를 통해 음성으로도 정보가 제공 됩니다.
  • 텍스트 뷰모드에서 Ctrl+Alt+3 단축키를 활용 하거나 화면 하단의 이미지뷰 버튼을 누르면 현재 원격제어 중인 원격지 PC의 화면 상태가 캡쳐되어 모바일 마이원리모트 화면에 이미지로 표시 됩니다.
  • 원격제어 중 언제든지 Ctrl+Alt+3 단축키를 누르면 원격지 PC의 화면을 전송 받을 수 있습니다.
  • 텍스트 뷰모드나 이미지 뷰모드에서 언제든 Ctrl+Alt+2 단축키를 누르거나 화면 하단의 FTP 뷰 버튼을 누르면 FTP 뷰모드로 전환 됩니다.
  • FTP 뷰모드에서는반드시 모바일 마이원리모트의 보이스오버를 활성화 한 상태에서 사용 합니다.
  • 가상키보드 뷰모드는 모바일 마이원리모트 홈 화면에서만 연결이 가능 합니다.
  • 가상키보드 뷰모드에서는 모바일 마이원리모트 디바이스의 보이스오버를 종료한 상태에서만 사용 가능 합니다.
